캐릭터 레벨 설계
구간별 등차 성장
  • 1레벨 4,300 XP → 60레벨 1,234,300 XP (구간별 등차 증가)
  • 60레벨 = 330시간 / 100레벨 = 1,200시간 (지역 완료 기준 앵커)
  • 레벨업 소요 시간이 급증하는 구간: 40레벨 이후 (하얀 숲~)

숙련도 시스템
티어 제한 경험치 획득

  • 자신의 숙련도 티어 이상 몬스터에서만 경험치 획득
  • 지역 클리어 시 해당 티어 자동 달성 — 레벨·숙련도 동기화 설계

전리품 제작
시간을 갈아넣으면 빠른 경로
  • 필드 사냥으로 전리품 수집 → 입장 제한 없음
  • 동일 티어 드랍률 T2: 8.33% / T3: 3.33% (티어별 설계)
  • 가공물 병행 요구 → 채집 활동 자연 유도
  • 하드코어 유저에게 가장 유리한 경로
🏰
던전 코어 제작
매일 조금씩 쌓이는 경로
  • 100% 확정 드랍 / 일 3회 제한 / 하루 45분
  • 코어 3~7개 랜덤 → 천장 보장형 설계
  • 라이트 유저 최적 경로 — 꾸준하면 반드시 완성
💀
보스 드랍
운빨 도파민 경로
  • 0.5~1% 확률 완제품 즉시 획득
  • 주력 경로 아님 — 보너스 도파민 용도
  • 예상치 못한 행운 → 강한 즉각적 성취감
⚙
설계 철학
경로 간 밸런스
  • 초반(T1~T3) 빠른 성장, 중후반(T4+) 무게감 증가
  • 세 경로 모두 보완적 — 어떤 선택도 불합리하지 않음
  • 사냥 속도 분당 10마리 / 채집 12회/분 기준
