# ⚡ X7 Enhancement Simulator

X7 게임의 **강화 시스템**을 시뮬레이션하는 웹 앱입니다.
델피나드 기운(천장 시스템) + 복구 메커니즘을 포함한 확률 기반 시뮬레이터.

🚀 **[라이브 데모 바로 가기](https://x7-enhance-sim-jj7yy4xzfsgaohw2bmudkv.streamlit.app/)**

---

## 기능

| 탭 | 설명 |
|----|------|
| 🎮 **인터랙티브** | 클릭으로 강화 직접 진행. 기운 게이지·복구 소모·예상 노강 실시간 표시 |
| 📊 **배치 통계** | N회 일괄 시뮬 → 시도 분포 히스토그램·분위수 테이블·7개 티어 비교 |
| 🔬 **몬테카를로 테이블** | 복구 포함 예상 노강 소모 기댓값 전체 테이블 + 히트맵 |

### 강화 시스템 규칙
- **+1**: 안전 강화 (100% 성공, 실패 없음)
- **+2 ~ +10**: 실패 시 강화 횟수 -1 차감 (단계 하락 없음)
- **델피나드 기운 (+2~+10)**: 실패마다 기운 충전, 100% 도달 시 다음 강화 **100% 보장** (천장)
  - +2,+3: 천장 5회 / +4: 천장 5회 / +5: 천장 7회 / +6: 천장 10회
  - +7: 천장 15회 / +8: 천장 25회 / +9: 천장 30회 / +10: 천장 40회
- **충전**: 강화 횟수 3회 소진 시 충전석 소모 후 횟수 복구

## 로컬 실행

```bash
# 1. 패키지 설치
pip install -r requirements.txt

# 2. 앱 실행
streamlit run enhance_app.py
```
