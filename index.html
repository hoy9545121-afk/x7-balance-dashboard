🗡
전리품 제작
시간을 갈아넣으면 빠른 경로
  • 필드 사냥으로 전리품 수집 → 입장 제한 없음
  • 동일 티어 드랍률 15% → 7마리당 1개, 긍정적 체감
  • 가공물 병행 요구 → 채집 활동 자연 유도
  • 하드코어 유저에게 가장 유리한 경로
🏰
던전 코어 제작
매일 조금씩 쌓이는 경로
  • 100% 확정 드랍 / 일 3회 제한 / 하루 45분
  • 코어 3~7개 랜덤 → 천장 보장형 설계
  • 라이트 유저 최적 경로 — 꾸준하면 반드시 완성
💀
보스 드랍
운빨 도파민 경로
  • 0.5~1% 확률 완제품 즉시 획득
  • 주력 경로 아님 — 보너스 도파민 용도
  • 예상치 못한 행운 → 강한 즉각적 성취감
⚙
설계 철학
경로 간 밸런스
  • 초반(T1~T3) 빠른 성장, 중후반(T4+) 무게감 증가
  • 세 경로 모두 보완적 — 어떤 선택도 불합리하지 않음
  • 사냥 속도 분당 20마리 / 채집 12회/분 기준
🎲
드랍률 정책
경로별 드랍 설계
동일 티어 전리품 15%
분당 3개 획득 (20마리 기준) · 7마리당 1개 — '자주 나온다' 체감
하위 티어 전리품 2%
희귀하지만 가끔 — 고레벨 구간 하위 재료 자연 수급
보스 완제품 드랍 0.5~1%
운빨 도파민 — 주력 경로 아님, 예상 밖 즉시 획득 쾌감

🛡 방어구 스킬 밸런스

🎯 설계 목적
스킬 레벨업의 유의미한 체감 — 모든 레벨에서 이전 대비 약 15% 성장, Lv.5 시 Lv.0 대비 2배 성능
티어 간 위계 확립 — 하위 티어 Lv.5보다 상위 티어 Lv.0이 더 높도록 설계
능력치 가치 평준화 — 각 옵션이 전투력에 기여하는 효율을 3~5%로 동일하게 맞춤
⚙ 성장 공식
성장 계수 1.15ⁿ (15% 복리 성장 적용)
라운딩 룰 — 0 또는 5 단위로 끝자리 보정, 직관적 수치 인지
전투 조율 상수 K=500 기준으로 능력치 1pt당 기여도 분석
🏆 티어 스케일링
1T (×1.0) — 기준. 기본 제작, 초반 성장의 핵심 도구
2T (×1.6) — 1T Lv.5와 유사 수준에서 시작 → 파밍 동기 부여
3T (×2.5) — 압도적 기본 성능 차이 → 지속적 던전 탐색 유도
